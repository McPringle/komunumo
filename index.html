You are a community leader? Komunumo makes your life a lot easier! Eveything you need to run your community in one place, for free! Komunumo is an open source community management software which assists you in all tasks managing a community like a user group.

With Komunumo you can manage your community members (including self-management), organize events (including registration), send out newsletters, manage your sponsors, and a lot more.

We aim to implement all key features of Meetup in Komunumo. We strive for full compatibility with the Fediverse to ensure smooth data exchange with Mastodon, Mobilizon, and other services using the ActivityPub protocol.

Komunumo is open source software licensed under the GNU Affero General Public License (AGPL) and free to use for everyone. It is being developed by a passionate group of people, who actively contribute to various communities in their free time. No one earns money with Komunumo. Komunumo belongs to the community!

For development we use a GitHub repository, where we track all issues and host a forum for discussions and support.

We use the Java programming language and the Vaadin framework for the server. For this website we just use static HTML 5 and GitHub Pages. Komunumo is still under heavy development – and any kind of help is highly appreciated!
