## Rich Text Formatting

**Example:**

```
You can create **bold text**, *italicized text*, and `inline code` with simple Markdown syntax.
You can also ~~strike through~~ text when needed.
```

**Result:**

You can create **bold text**, *italicized text*, and `inline code` with simple Markdown syntax.
You can also ~~strike through~~ text when needed.
